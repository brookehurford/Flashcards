Variable naming conventions

Variable naming conventions

  • Variables should begin with a letter (they can also begin with $ or _ but these symbols are not advisable...stick with the letters!).
  • Variables are case sensitive (myNumber is a different variable than myNUMBER).
  • Use clear, descriptive names that describe the value being stored like myNumber.

+=

+=

A shortcut for changing variable value. The two following examples mean the exact same thing:

var myVariable+=5

var myVariable = myVariable + 5

Function

Function

JavaScript functions are defined with the function keyword, followed by a name, followed by parentheses ().

Method

Method

JavaScript methods are the actions that can be performed on objects. A JavaScript method is a property containing a function definition.

Argument to a Method

Argument to a Method

Goes inside the parenthesis of a method. Arguments provide a bit more information to methods to help them know what they're supposed to do.

String

String

Method Chaining

Method Chaining

Escape

Commenting Code
